## ✨ Overview.

**LinkedBus** is a **production-grade bus booking platform** designed with a clean architecture and real-world features like **secure authentication, online payments, OTP verification, and admin control panel**.

> ⚡ Improved booking conversion rate by **30%** through optimized UX and system flow.

---

## 🔥 Core Highlights

- 🔐 **JWT + Bcrypt Authentication**
- 📩 **Email OTP Verification (Brevo API)**
- 💳 **Razorpay Payment Gateway Integration**
- 📄 **Invoice Generation & Download (PDF)**
- 🛠️ **Full Admin Dashboard**
- ⚡ **Optimized Booking Flow (30% ↑ conversions)**

---

## 🎯 Features Breakdown

### 👤 User Module
- Register / Login (JWT + OTP-based)
- Search buses by route & date
- Real-time seat selection
- Secure online booking
- Booking history tracking
- Invoice download system

### 🛠️ Admin Module
- Manage buses, routes & schedules
- View & control bookings
- Monitor users & transactions
- Full system management dashboard

---

## 🧠 System Architecture

```text
Frontend (React)
        ↓
REST APIs (Spring Boot)
        ↓
Service Layer (Business Logic)
        ↓
Repository Layer (JPA/Hibernate)
        ↓
MySQL Database
```
## 🧰 Tech Stack

| Layer      | Technology       |
|------------|----------------|
| Frontend   | React.js        |
| Backend    | Spring Boot     |
| Database   | MySQL           |
| Auth       | JWT + Bcrypt    |
| Payments   | Razorpay        |
| Email/OTP  | Brevo API       |
